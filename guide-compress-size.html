如何把图片压缩到指定大小（100KB / 50KB / 20KB 以内）

很多场景对上传的图片有明确的大小限制：考试报名要求证件照不超过 20KB，招聘网站要求简历照片小于 100KB，政务/银行系统常常限制在 50KB 以内。图片超过限制就传不上去，是非常常见的烦恼。

这篇教程教你用 轻图压缩 把图片精确压到指定大小以内，全程在浏览器本地完成，图片不上传服务器。

最快的方法：用「压缩到指定大小」一键压

  1. 打开 轻图压缩工具，把图片拖进去（或点击选择，支持一次多张）。
  2. 在「压缩到指定大小（KB，0=不限）」里填上你要的目标，比如报名要求 ≤20KB 就填 20。
  3. 点「开始压缩」。工具会自动二分调节质量、必要时缩小尺寸，把文件压到目标以内。
  4. 压好后点「下载」保存即可。

填写了目标大小后，上面的「压缩质量」会被忽略——因为工具会自动帮你找到刚好满足大小的质量，不用你手动反复试。

压不到目标大小怎么办？

如果图片信息量很大（比如高清风景照要压到 20KB），仅靠降低质量可能不够，可以再配合下面两招：

  • 降低「最大宽度」：把尺寸缩小（例如填 800 或 500），体积会大幅下降。证件照、头像本来就不需要很大。
  • 输出选 JPG 或 WebP：这两种是有损格式，比 PNG 更容易压到很小。PNG 是无损的，主要靠缩小尺寸来减小体积。

如果实在压不到，工具会显示「已压到最小，仍超目标」，并给出它能做到的最小体积——这时把尺寸再调小一些通常就能达标。

不同场景的推荐目标值

  • 证件照上传（考试/报名）：常见要求 ≤ 20–50KB，格式 JPG。
  • 简历 / 求职头像：≤ 100KB 一般够用，清晰度也不错。
  • 网页配图 / 商品图：100–300KB，兼顾清晰与加载速度。
  • 论坛 / 社交分享：200–500KB，基本随意。

具体以对方系统的要求为准，拿不准就按它给的上限填，宁可小一点。

常见问题

压缩会把我的照片上传到你们服务器吗？

不会。轻图压缩基于浏览器的 Canvas 技术，所有处理都在你本地设备完成，图片不会被上传或保存，证件照、身份证照片也能放心处理。

为什么有的图压到目标后看起来糊了？

把文件压得很小，本质是丢弃图像细节，体积越小、画质越低，这是所有压缩工具的共性。建议在“能满足大小要求”的前提下，尽量给大一点的目标值。
